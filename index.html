Formulários

Formulários são usados para coletar dados dos usuários. Eles podem incluir campos de texto, botões, caixas de seleção, etc. Aprendemos sobre Tags Form, Tag Input,Checkbox e Radio, Button, Select e Textarea.

→Tags Form

A tag <form> é usada para criar formulários. Ela pode conter vários tipos de campos de entrada.

→Tag Input

A tag <input> é usada para criar campos de entrada de dados. Existem vários tipos, como texto, senha, email, etc.

→Checkbox e Radio

Checkboxes permitem que os usuários selecionem várias opções, enquanto os botões de rádio permitem apenas uma seleção entre várias opções.

→Button

A tag <button> é usada para criar botões que podem ser clicados para enviar formulários ou executar ações.

→Select

A tag <select> cria uma lista suspensa de opções. Os usuários podem escolher uma opção da lista.

→Textarea

A tag <textarea> é usada para criar áreas de texto onde os usuários podem inserir texto longo.

Estrutura e Formatação

A estrutura do HTML é composta por elementos como cabeçalhos, parágrafos, listas e links. A formatação pode ser feita com CSS.

→Estrutura Básica do HTML

A estrutura básica de um documento HTML inclui a declaração do tipo de documento, a tag <html>, e as seções <head> e <body>.

→Tags Importantes

Algumas tags importantes incluem <h1> a <h6> para cabeçalhos, <p> para parágrafos, e <a> para links.

→Formatação de Texto

O texto pode ser formatado usando tags como <strong> para negrito, <em> para itálico, e <u> para sublinhado.

→Listas

Listas podem ser ordenadas (<ol>) ou não ordenadas (<ul>). Itens de lista são definidos com a tag <li>.

→Tabelas

Tabelas são criadas com a tag <table> e podem conter linhas (<tr>), células de cabeçalho (<th>) e células de dados (<td>).

→Links e Imagens

Links são criados com a tag <a> e imagens com a tag <img>. Os links podem ser internos ou externos, e as imagens podem ter atributos como src e alt.

→Comentários

Comentários em HTML são feitos com a sintaxe <!-- comentário -->. Eles não são exibidos no navegador, mas ajudam na organização do código.

→Meta Tags

Meta tags são usadas dentro da seção <head> para fornecer informações sobre o documento, como descrição, palavras-chave e autor.

→Divs e Spans

A tag <div> é usada para agrupar blocos de conteúdo, enquanto a tag <span> é usada para agrupar texto em linha. Ambas são úteis para aplicar estilos com CSS.

→Fieldset

A tag <fieldset> é usada para agrupar elementos de formulário relacionados, enquanto a tag <legend> fornece um título para o grupo.

→Iframe

A tag <iframe> é usada para incorporar outro documento HTML dentro do documento atual. É útil para exibir conteúdo externo, como vídeos ou mapas.

→Cores

As cores podem ser definidas em HTML usando nomes de cores, valores hexadecimais ou valores RGB. Elas são aplicadas através de CSS.

Mídias

HTML permite a incorporação de imagens, vídeos e áudio, enriquecendo a experiência do usuário.

→Tag Img

A tag <img> é usada para inserir imagens em uma página web. Ela requer o atributo src para especificar o caminho da imagem e o atributo alt para fornecer uma descrição alternativa.

→Tag Video

A tag <video> é usada para incorporar vídeos. Ela pode conter atributos como controls para exibir controles de reprodução, autoplay para iniciar automaticamente e loop para reproduzir em loop.

→Tag Audio

A tag <audio> é usada para incorporar áudio. Assim como a tag de vídeo, ela pode ter atributos como controls, autoplay e loop.

→Tag Track

A tag <track> é usada dentro da tag <video> ou <audio> para fornecer legendas ou descrições de áudio. Ela pode ter atributos como kind, src e srclang.

→Tag Source

A tag <source> é usada dentro das tags <video> e <audio> para especificar diferentes formatos de mídia. Isso permite que o navegador escolha o formato mais adequado para reprodução.

→Tag Inframe

A tag <iframe> é usada para incorporar outro documento HTML dentro do documento atual. É útil para exibir conteúdo externo, como vídeos do YouTube ou mapas do Google Maps.

Tabelas

Tabelas são usadas para organizar dados em linhas e colunas, facilitando a visualização de informações.

→Tag Table

A tag <table> é usada para criar uma tabela. Ela contém outras tags como <tr> para linhas, <th> para células de cabeçalho e <td> para células de dados.

→Tag tr

A tag <tr> é usada para definir uma linha em uma tabela. Cada linha pode conter várias células de dados ou cabeçalho.

→Tag td e th

A tag <td> é usada para definir uma célula de dados em uma tabela, enquanto a tag <th> é usada para definir uma célula de cabeçalho. As células de cabeçalho geralmente são exibidas em negrito e centralizadas.

→Tag tbody, thead e tfoot

As tags <tbody>, <thead> e <tfoot> são usadas para agrupar o corpo, o cabeçalho e o rodapé de uma tabela, respectivamente. Isso ajuda a organizar melhor os dados e a aplicar estilos específicos.

→Estilização de Tabelas

As tabelas podem ser estilizadas usando CSS para melhorar a aparência. Isso inclui definir bordas, cores de fundo, espaçamento entre células e fontes.

→Exemplo de Tabela

Abaixo está um exemplo simples de uma tabela em HTML:

<table>           
    <thead>
        <tr>
            <th>Nome</th>
            <th>Idade</th>      
        <th>Cidade</th>
        </tr>
    </thead>
    <tbody>
                
        <tr>
            <td>João</td>
            <td>30</td>
            <td>São Paulo</td>
        </tr>
        <tr>
            <td>Maria</td>
            <td>25</td>
            <td>Rio de Janeiro</td>
        </tr>
        <tr>  
            <td>Pedro</td>
            <td>28</td>
            <td>Belo Horizonte</td>
        </tr>
    </tbody>
</table>

HTML Semântico

HTML semântico usa tags que têm significado claro, melhorando a acessibilidade e SEO das páginas web.

→Importância do HTML Semântico

O HTML semântico ajuda os navegadores e mecanismos de busca a entenderem melhor o conteúdo da página. Isso melhora a acessibilidade para pessoas com deficiência e otimiza o SEO.

→Tags Semânticas

Algumas tags semânticas importantes incluem:

  • <header> - Define o cabeçalho da página ou seção.
  • <nav> - Define uma seção de navegação.
  • <main> - Define o conteúdo principal da página.
  • <article> - Define um conteúdo independente, como um post de blog.
  • <section> - Define uma seção genérica de conteúdo.
  • <aside> - Define conteúdo relacionado, como barras laterais.
  • <footer> - Define o rodapé da página ou seção.
  • <figure> - Define conteúdo ilustrativo, como imagens com legendas.
  • <figcaption> - Define a legenda de uma figura.
  • <time> - Define uma data ou hora.
  • <mark> - Define texto destacado.
  • <address> - Define informações de contato do autor ou proprietário do documento.

→Acessibilidade

O uso de tags semânticas melhora a acessibilidade, pois leitores de tela e outros dispositivos assistivos podem interpretar melhor o conteúdo. Isso é especialmente importante para pessoas com deficiência visual ou cognitiva.

→SEO

O HTML semântico também melhora o SEO, pois os mecanismos de busca podem entender melhor a estrutura e o conteúdo da página. Isso pode resultar em melhores classificações nos resultados de pesquisa.

→Web Scraping

O HTML semântico facilita o web scraping, pois as tags bem definidas permitem que os scripts identifiquem e extraiam informações de maneira mais eficiente.

→Article, Blockquote e Q

A tag <article> é usada para definir um conteúdo independente, como um post de blog. A tag <blockquote> é usada para citações longas, enquanto a tag <q> é usada para citações curtas em linha.
